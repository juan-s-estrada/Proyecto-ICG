En ente programa, los archivos son leidos por drag-and-drop. Arrastra el icono del 
archivo en la ventana y se abriran.

Incluyo archivos OBJ y OFF. Los archivos OBJ con "/" no son parseados correctamente
lo que ocasiona que el programa se rompa dependiendo del archivo. Archivos obj
sin "/" como los suministrados funcionan perfectamente.

Recuerda dar click a select para seleccionar algun modelo ya cargado.
